# Credit Analysis Tool

A credit risk analysis platform for Norwegian companies, combining 
live financial data with CFA-based methodology and machine learning.

## Features
- Single company credit analysis with real-time data from 
  Brønnøysund Register and Roaring API
- Rule-based credit scoring grounded in CFA Fixed Income framework
- Random Forest ML model for probability of default (PD) estimation
- Altman Z-score classification
- Portfolio monitoring of 1,000 Norwegian companies
- Basel III ROE calculator for a NOK 10bn asset finance portfolio 
  across four segments (hard assets, IT, agriculture, medical equipment)

## Tech Stack
Python · Streamlit · scikit-learn · Plotly · Roaring API · 
Brønnøysund Register API
